# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| 1.0.x   | :x:                |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

If you've discovered a security vulnerability in CertifyPro, please **do not opening a public issue**. Instead, report it privately to:

**Security Team:** security@certifypro.com

Please include:
- **Type of issue** (e.g., SQL injection, XSS, etc.)
- **Detailed steps to reproduce**
- **Affected component** (Client, Server, or Database)
- **Potential impact**

We aim to acknowledge all reports within 48 hours and provide a resolution within 14 days.
